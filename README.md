# Реализация алгоритмов Флёри и Дейкстры
Работу выполнил студент группы 20300 Миронов Д.

# Алгоритм Флёри
Постановка математической задачи и метод ее решения.
Построение Эйлерова цикла на графе посредством алгоритма Флери.

Листинг программы и описание.
На входе имеется связный граф с вершинами четных степеней. Посредством удаления ребер один за другим мы проходим по всем ребрам и возвращаемся в начальную вершину (цикл). На выходе последовательность ребер (сам цикл).


# Алгоритм Дейкстры

Постановка задачи и ее метод решения.
Задача - поиск кратчайшего пути в графе с использованием алгоритма Дейкстры. 

Листинг программы.
На входе имеется взвешенный неориентированный граф (весы графа должны быть неотрицательными).
На выходе ряд чисел (кратчайшие пути) к каждой вершине, начиная с начальной.
